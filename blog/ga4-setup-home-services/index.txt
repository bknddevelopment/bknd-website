|----------------|-------|
| phone_number | {{Click URL}} |
| page_location | {{Page URL}} |
| page_title | {{Page Title}} |</p><p class="text-[#86868B] mb-6 leading-relaxed">Set the trigger to "Trigger - Phone Link Click".</p><p class="text-[#86868B] mb-6 leading-relaxed">Name it "GA4 - Event - Phone Click" and save.</p><p class="text-[#1D1D1F] font-semibold mb-4">Step 3: Test It</p><p class="text-[#86868B] mb-6 leading-relaxed">Click Preview in GTM. - <strong class="text-[#1D1D1F]">Trigger Type</strong>: Page View
- <strong class="text-[#1D1D1F]">This trigger fires on</strong>: Some Page Views

    });
  });
